# OSEF Framework Simulation

## Overview
OSEF (Operational Stability Envelope Framework) is an **advisory-only real-time aviation safety monitoring system**.  
This simulation demonstrates:

- Creative Chaos Zone (CCZ) alerts
- Limit cycle detection
- Stability envelope monitoring

**Important:** This simulation is advisory-only and does **not control any aircraft**.

## Requirements
- Python 3.9+
- Libraries: numpy, matplotlib, XPlaneConnect (optional for X-Plane integration)

Install dependencies:
```bash
pip install numpy matplotlib XPlaneConnect
